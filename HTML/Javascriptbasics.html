Numbers

Can be used to complete Math Calculations

  • BIDMAS principle applied
  • Modulo (%) / Calculates the remainder. 19 % 4 = 3
  • Exponentiation (**) Creates an Indicy. 2**3 = 8
  • NaN - Represents something that is not a Number. But is treated as a number
Examples:
0/0 = NaN
1 + NaN = NaN

Variables

Variables are like labels for values

  • We can store a value and give it a name so that we can;
    • Refer back to it
    • Use that value to do... stuff
    • Or change it later
Basic Syntax
let year = 1985;
const DaysInWeek = 7; (const cannot be changed)
var name = Joe; (the old way of labelling a variable. - no longer used)
Updating Variables

Variable += 1

Variable++ = An increase by 1
