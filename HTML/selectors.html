Selectors

A CSS selector is the part of a CSS rule that describes what elements in a document the rule will match. The matching elements will have the rule's specified style applied to them.

Selector Name Description
* Universal selector Selects everything on a page
h1 Element selector Selects a specific element wether thats a h1, li, or table
#id ID selector Singles out an individual element. Cannot have more than one on a page
.class class selector The CSS class selector matches elements based on the contents of their class attribute.
li a descendant selectors Only anchor tags inside of an li will be styled. Not li's
h2 + p adjacent selector Select only the element that immediately proceeds another element, for example the first p after h2's
div > li child selector Select only li's that are direct children / decsendants of a div

Psuedo classes

Pseudo class selectors targets elements depending on their state rather than on information from the document tree. For example, the selector a:visited applies styles only to links that the user has already followed.

Psuedo elements

A psuedo element is a keyword added to a selector that lets you style a specific part of the selected element(s). For example, ::first-line can be used to change the font of the first line of a paragraph.

Specificity

Specificity is the means by which browsers decide which CSS property values are the most relevant to an element and, therefore, will be applied. Specificity is based on the matching rules which are composed of different sorts of CSS selectors.
